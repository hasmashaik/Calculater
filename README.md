Simple Calculator
This is a basic calculator made with HTML, CSS, and JavaScript.

What it Does
Lets you add, subtract, divide numbers

Has a clean and simple design

Shows results when you click =

Clears input when you click C

Handles decimal numbers

How to Use
Open the HTML file in a web browser.

Click the buttons to enter numbers and symbols.

Click = to see the result.

Click C to clear the screen.

Notes
One button in the layout is empty — you can add something like * for multiplication there.

It uses JavaScript’s eval() to calculate expressions.
